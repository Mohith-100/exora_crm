⚡ Score Engine Settings
All scoring rules are saved in your database — changes apply to future scoring runs immediately
BASE SCORE RULES
Points awarded from Google Maps data (rating, reviews, presence signals)
GAP BOOST RULES
Bonus points added when a gap is detected on the school's website
🔍 HOW GAP DETECTION WORKS
CRM / Enquiry
Scans for HubSpot, LeadSquared, Zoho, Meritto scripts in webpage code
Fee Payment
Looks for Razorpay, Paytm, Stripe, PayU payment gateway scripts
Mobile App
Checks for Play Store / App Store links in the page
HTTPS / SSL
Checks if the website URL uses HTTPS or not
LMS
Scans for Moodle, Canvas, Teachable, LearnPress etc.
Live Chat
Detects Tawk.to, Tidio, WhatsApp, Zendesk, Freshchat
Admission Portal
Looks for keywords: "apply online", "registration form"
ERP / Attendance
Detects Fedena, Entab, MyClassboard, SchoolPad
⚠️ Limitation: Only the school's public website is scanned. If a tool is used internally (separate app/portal not embedded in the website), it won't be detected. Gap = "not found on website" not "doesn't exist".
